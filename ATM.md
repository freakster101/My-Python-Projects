# ATM Project

A simple terminal-based ATM system built with Python.

## Features

- PIN verification
- Withdraw money
- Deposit money
- Check balance
- Transaction history
- Menu system using loops

## Concepts Practiced

- Functions
- Dictionaries
- Lists
- Loops
- Conditional statements
- User input
- Basic project structure

## Data Structure Used

```python
accounts = {
    "name": "Jagdish",
    "pin": "1994",
    "balance": 5000,
    "transaction": []
}
```

- Difference between lists and dictionaries
- Using dictionaries inside lists
- Updating values inside dictionaries
- Using `.append()` correctly
- Returning values from functions
- Structuring a small Python project
